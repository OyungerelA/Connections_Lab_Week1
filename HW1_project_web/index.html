Space Survival

a desktop game

About Game Description Contact

Space Survival is an interactive one-player game where an astronaut is positioned at the bottom of the screen and can be moved horizontally within the screen dimensions. Two types of objects descend from sky randomly with a certain speed: meteors which the astronaut must dodge and oxygen that it will try to catch to survive. Multiple objects fall at a given time and the astronaut must be moved accordingly to dodge or catch objects. The game score will be incremented by 1 whenever the astronaut catches a regular oxygen and by 10 when it catches a gold oxygen. If the astronaut fails to dodge a meteor and is hit by it, the game ends and the score will be displayed showing the score and the highest score. Afterwards, the game will restart upon pressing the UP button.
